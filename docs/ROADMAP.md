# Roadmap de desenvolvimento

## Etapa 1: base do projeto

- Criar estrutura de pastas.
- Criar arquivos iniciais.
- Configurar Vite.
- Configurar variaveis de ambiente.

## Etapa 2: Supabase

- Instalar dependencias.
- Configurar cliente Supabase.
- Testar conexao com o projeto Supabase.

## Etapa 3: autenticacao

- Criar tela de login.
- Criar logout.
- Identificar usuario logado.
- Buscar papel do usuario.

## Etapa 4: musicas

- Listar musicas.
- Cadastrar musica.
- Editar musica.
- Converter cifra para ChordPro.

## Etapa 5: repertorios

- Listar repertorios.
- Cadastrar repertorio.
- Editar repertorio.
- Associar musicas ao repertorio.
- Ordenar musicas dentro do repertorio.

## Etapa 6: permissoes

- Admin gerencia tudo.
- Editor cadastra e edita musicas/repertorios.
- Musico visualiza musicas e repertorios.
